Securing Passwords

All passwords for all accounts have been updated to 64 alphanumerical stochastic characters. They've been hashed multiple times on top of each hash and each character has been stochastically determined from the hash using a unique function I developed.

2 Factor Authentication

In the event some developer doesn't salt and hash passwords (like instatutors.org) I use 2 factor authentication with a physical Solo Key that uses FIDO U2F developed by Google and Yubico.
